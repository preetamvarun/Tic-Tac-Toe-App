## 4. Getting Started 🚀

#### [ Setup Flutter](https://flutter.io/setup/)

#### Clone the repo

```sh
$ git clone https://github.com/preetamvarun/Tic-Tac-Toe-App.git
$ cd Tic-Tac-Toe-App/
```

#### Install dependencies 
```sh
$ flutter pub get
```

#### Run app
```sh
$ flutter run
```
