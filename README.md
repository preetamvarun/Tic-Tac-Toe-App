## Getting Started 🚀

#### 1. [ Setup Flutter](https://flutter.io/setup/)

#### 2. Clone the repo

```sh
$ git clone https://github.com/preetamvarun/Tic-Tac-Toe-App.git
$ cd Tic-Tac-Toe-App/
```

#### 3. Install dependencies 
```sh
$ flutter pub get
```

#### 3. Run app
```sh
$ flutter run
```
